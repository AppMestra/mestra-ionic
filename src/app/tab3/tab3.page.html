Sobre o Mestra
Informações do Projeto

O Mestra é uma aplicação completa que integra diferentes tecnologias para fornecer uma solução robusta de comunicação e processamento de dados.

Tecnologias Utilizadas
  • Frontend: Ionic + Angular
  • Backend: API Golang
  • Bot: Node.js + Venom
  • UI: Bootstrap + Ionic
  • WhatsApp: Integração
  • Banco: PostgreSQL/MySQL
Funcionalidades
  • Processamento de mensagens em tempo real
  • Integração com WhatsApp via Venom Bot
  • API REST para comunicação entre serviços
  • Interface responsiva e moderna
  • Gerenciamento de entradas e mensagens
